# Diagnostic des problèmes de batterie et d'alimentation

## Symptômes fréquents
Les problèmes de batterie peuvent inclure :
- batterie qui se décharge rapidement ;
- produit qui ne charge plus ;
- charge bloquée à un certain pourcentage ;
- extinction soudaine ;
- produit qui ne démarre pas sans chargeur ;
- surchauffe pendant la charge.

## Causes possibles
Les causes possibles sont :
- batterie usée ;
- chargeur défectueux ;
- connecteur de charge endommagé ;
- carte d'alimentation défectueuse ;
- consommation anormale par un logiciel ;
- problème de calibration de batterie ;
- surchauffe interne.

## Actions recommandées
L'agent SAV doit :
1. vérifier si le chargeur est officiel ;
2. tester un autre chargeur compatible ;
3. contrôler l'état du connecteur de charge ;
4. vérifier l'âge du produit ;
5. consulter la date de fin de garantie ;
6. recommander un test de santé batterie.

## Garantie
L'usure normale de la batterie peut ne pas être couverte par la garantie. Cependant, une panne prématurée peut être prise en charge si le produit est encore sous garantie.

## Recommandation IA
Si le client indique une extinction soudaine ou une charge instable, l'IA doit recommander un contrôle batterie, chargeur et connecteur d'alimentation.
