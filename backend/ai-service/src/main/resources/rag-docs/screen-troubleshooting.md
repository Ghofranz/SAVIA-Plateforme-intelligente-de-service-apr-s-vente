# Diagnostic des problèmes d'écran

## Symptômes fréquents
Les problèmes d'écran peuvent se manifester par :
- écran noir ;
- écran qui clignote ;
- affichage très sombre ;
- lignes verticales ou horizontales ;
- absence d'affichage après démarrage ;
- image visible sur écran externe mais pas sur l'écran principal.

## Causes possibles
Les causes les plus fréquentes sont :
- câble écran mal connecté ou endommagé ;
- dalle LCD défectueuse ;
- carte graphique instable ;
- pilote graphique corrompu ;
- surchauffe du processeur graphique ;
- problème d'alimentation de l'écran ;
- mise à jour système incorrecte.

## Actions recommandées
L'agent SAV doit vérifier :
1. si le produit démarre normalement ;
2. si un écran externe fonctionne ;
3. si le rétroéclairage est actif ;
4. si le problème apparaît avant ou après le chargement du système ;
5. si le client a récemment fait une mise à jour ;
6. si le produit a subi un choc.

## Priorité
Un écran totalement noir doit être traité avec priorité élevée si le produit est inutilisable.

## Recommandation IA
Si la description mentionne un écran noir après quelques minutes, l'IA doit envisager une surchauffe, un problème de pilote graphique ou une panne matérielle de la dalle.
