# Diagnostic des problèmes réseau

## Symptômes fréquents
Les problèmes réseau peuvent inclure :
- Wi-Fi qui ne se connecte pas ;
- connexion instable ;
- débit très lent ;
- Bluetooth indisponible ;
- carte réseau non détectée ;
- perte de connexion après mise en veille.

## Causes possibles
Les causes les plus fréquentes sont :
- pilote réseau corrompu ;
- carte Wi-Fi défectueuse ;
- problème de configuration IP ;
- routeur incompatible ou instable ;
- antenne interne déconnectée ;
- mise à jour système ayant modifié la configuration réseau.

## Actions recommandées
L'agent SAV doit :
1. demander si le problème existe sur plusieurs réseaux ;
2. vérifier si les autres appareils se connectent normalement ;
3. recommander une réinitialisation réseau ;
4. vérifier le gestionnaire de périphériques ;
5. tester la connexion Ethernet si disponible ;
6. vérifier les mises à jour du pilote réseau.

## Priorité
Un problème réseau est critique si le produit est utilisé pour le travail à distance ou si aucune connexion alternative n'est possible.

## Recommandation IA
Si le client indique une connexion instable après une mise à jour, l'IA doit suspecter un problème de pilote ou de configuration réseau.
